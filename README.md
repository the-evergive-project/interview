## 3. System Archiceture (30 mins)

1. We want to launch the app. What kind of infrastructure do you recommend?
    1. What kind of db?
    2. What kind of nodes?
    3. How do you handle infrastructure? (IaC, old-school vms, cloud manamagent, serverless)
    4. How do you handle security
    5. How do you handle scalabilty

2. Now that the core product is live, secure and scalable, what are the supporting platforms that you would recommend (e.g. monitorability, observability, data warehousing etc)?
    1. What kind of solutions do you have in mind?
    2. How would you rollout these solutions?
    3. Consider security, scalability, cost and complexity

**Notes**:
* If CI/CD is not mentioned: how do we deploy the whole thing?
* If tests not mentioned: what kind of tests would you write for this app?
